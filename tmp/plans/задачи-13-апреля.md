## Задача 1: Тултип для описания параметра

**Проблема:** На странице модели (`/v2/model/[id]`) у параметров с описанием есть иконка `?`, но при наведении показывается примитивный браузерный `title`-тултип.

**Решение:** Заменить на стилизованный CSS-тултип через Tailwind `group/group-hover`.

### Шаги:
1. **Создать компонент `Tooltip.tsx`** — `frontend/src/components/Tooltip.tsx`
   - Pure CSS через `group` + `absolute` позиционирование
   - Поддержка dark mode, `max-w-xs`, плавное появление
   - Без JS-состояния, без зависимостей

2. **Обновить `IndexCriterionCard.tsx`** — строки 39-46
   - Заменить `<span title={criterion_description}>` на `<Tooltip text={criterion_description}>`
   - Убрать атрибут `title`

### Файлы:

## Задача 2: Фикс каскадного удаления параметров

**Проблема:** Удаление критериев из инлайна методики каскадно удаляет все `ModelRawValue` (значения параметров моделей). Удаление методики удаляет ВСЕ данные.

**Цепочка:** `MethodologyVersion` →(CASCADE)→ `Criterion` →(CASCADE)→ `ModelRawValue`

**Решение:** Изменить `ModelRawValue.criterion` с `CASCADE` на `SET_NULL`, добавить `criterion_code` для сохранения идентичности.

### Шаги:

1. **Изменить модель `ModelRawValue`** — `backend/catalog/models.py`
   - FK `criterion`: `on_delete=CASCADE` → `on_delete=SET_NULL`, `null=True, blank=True`
   - Добавить поле `criterion_code = CharField(max_length=50, db_index=True)`
   - Добавить `save()` — автозаполнение `criterion_code` из `criterion.code`
   - Обновить `__str__()` для nullable criterion
   - Обновить unique constraint: условный `(model, criterion)` при `criterion IS NOT NULL` + условный `(model, criterion_code)` при `criterion IS NULL`

2. **Создать и применить миграции** — `backend/catalog/migrations/`
   - Schema migration: новое поле + изменение FK + новые constraints
   - Data migration: backfill `criterion_code` из существующих `criterion.code`

3. **Фикс scoring engine** — `backend/scoring/engine/computation.py`
   - Добавить `criterion__isnull=False` в фильтр raw_values

4. **Фикс migration service** — `backend/catalog/services/raw_values_migration.py`
   - Фильтр `criterion__isnull=False` + передача `criterion_code` при создании

5. **Фикс сериализаторов** — `backend/catalog/serializers.py`
   - `RawValueSerializer` (строки 46-47): `criterion.code` и `criterion.name_ru` упадут при `criterion=None` — добавить fallback на `criterion_code`
   - `ACModelListSerializer._get_scores_cache` (строка 113): фильтр null criterion

6. **Защита от удаления активной методики** — `backend/methodology/admin/methodology_version.py`
   - Добавить проверку в `delete_model()`: запрет удаления активной методики

### Файлы:
- `backend/catalog/models.py`
- `backend/catalog/serializers.py`
- `backend/scoring/engine/computation.py`
- `backend/catalog/services/raw_values_migration.py`
- `backend/methodology/admin/methodology_version.py`
- `backend/catalog/migrations/` (2 новых файла)
